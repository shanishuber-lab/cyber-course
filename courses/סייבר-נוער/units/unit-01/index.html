חלק 1

מהי רשת מחשבים?

רשת מחשבים = קבוצה של מכשירים שמחוברים אחד לשני כדי להעביר מידע. ▾ הסבר מלא
רשת יכולה להיות קטנה — כמו רשת ביתית שמחברת מחשב, טלפון ומדפסת — או ענקית, כמו האינטרנט שמחברת מיליארדי מכשירים ברחבי העולם.

כדי שמחשבים יוכלו להבין אחד את השני, הם משתמשים בפרוטוקולים — כללים מוסכמים לתקשורת. לדוגמה: HTTP לטעינת אתרים, ו-IP לזיהוי כתובת של כל מכשיר ברשת.
דוגמה מהחיים — קבוצת WhatsApp
📱 כל אחד בקבוצה = מכשיר
💬 ההודעות = מידע
👥 הקבוצה עצמה = רשת
רכיבי הרשת

לחצו על כל כותרת לפרטים נוספים

💻
מארח Host
▾
מכשיר שמחובר ישירות לרשת ויכול לשלוח ולקבל מידע. דוגמאות: מחשבים, שרתים, מדפסות רשת, מצלמות רשת. לכל מארח יש כתובת IP ייחודית ברשת.

השוואה: מדפסת ביתית שמחוברת למחשב אחד בלבד היא "ציוד היקפי" ולא מארח עצמאי. רק אם תחובר ישירות לרשת — תהפוך למארח.
🔀
אביזרי רשת Router · Switch · Hub
▾
נתב (Router) — מחבר את הבית לאינטרנט, מנהל את תנועת הנתונים.
מתג (Switch) — מחבר מחשבים בתוך אותה רשת מקומית. יעיל יותר מרכזת.
רכזת (Hub) — מפיץ מידע לכל המחשבים ברשת, גם לאלה שלא ביקשו.
📡
מדיה כבל / Wi-Fi
▾
האמצעים שבהם עובר המידע ברשת:
רשת קווית — כבל נחושת או סיב אופטי. מהירה, יציבה, פחות פגיעה להאזנה.
רשת אלחוטית (Wi-Fi) — גלי רדיו. גמישה ונוחה, אך פגיעה יותר להפרעות ולהאזנות.

הכתובת שרואים אותך לפיה באינטרנט. משתנה!
▾ פרטים
ייחודית בעולם. בדרך כלל כל המכשירים בבית משתמשים באותה כתובת חיצונית. הנתב הוא שמתרגם בין הכתובות הפנימיות לכתובת החיצונית — תהליך שנקרא NAT.
